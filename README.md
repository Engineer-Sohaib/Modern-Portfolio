# 🚀 Modern Portfolio

A sleek, modern, and fully responsive developer portfolio built to showcase projects, skills, experience, and professional achievements with stunning UI animations and engaging user interactions.

![Portfolio Preview](https://res.cloudinary.com/dpx3gst4q/image/upload/v1783954349/Portfolio-Preview_a4dux4.png)

## ✨ Features

- 🎨 Modern & Clean UI Design
- 📱 Fully Responsive Across Devices
- ⚡ Smooth Animations & Transitions
- 🌙 Dark Theme Experience
- 🖼️ Interactive Project Showcase
- 💼 Professional About Section
- 🛠️ Skills & Technologies Display
- 📞 Contact Section
- 🚀 Performance Optimized
- 🔍 SEO Friendly

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Engineer-Sohaib/Modern-Portfolio.git
```

### Navigate to Project

```bash
cd Modern-Portfolio
```

### Run Locally

Simply open:

```bash
index.html
```

Or use VS Code Live Server for the best development experience.
